# 🧪 QA Testes - Sistema Bancário (Simulado)

Projeto prático de Quality Assurance simulando testes em um sistema bancário, com foco em qualidade, confiabilidade e prevenção de falhas.

---

## 🎯 Objetivo
Demonstrar na prática habilidades em:
- Testes funcionais  
- Testes de regressão  
- Criação de cenários de teste  
- Identificação e reporte de bugs  

---

## 🧪 Escopo do sistema
Simulação de funcionalidades bancárias:
- Login de usuário  
- Transferência de valores  
- Consulta de saldo  

---

## 📂 Estrutura do projeto

- 🔐 Cenários: cenarios-de-teste.md  
- 📋 Casos de teste: casos-de-teste.md  
- 🐞 Bugs: bugs-reportados.md  

---

## 🚀 Resultado

Projeto desenvolvido para demonstrar experiência prática em QA, com foco em ambientes críticos e sistemas financeiros.
